Spider Online Bibliography Database - MUD version.

The Spider Bibliography database is a collection of educational technology references, formatted as a BibTeX database. There are two search interfaces:

1. BibLook Interface: It will return Bibtex formatted entries, that can be used with the HTML2Latex Translator

This interface is experimental. It is based upon Jeff Erickson's and Bill Jones' biblook and bibindex programs. The WWW gateway is based on Stephen Spencer's (bibadmin@siggraph.org) code for the ACM SIGGRAPH Online Bibliography Database.

2. Bibsearch Interface: produces formatted output and clickable URLs.
