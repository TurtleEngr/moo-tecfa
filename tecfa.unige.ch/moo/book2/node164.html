16.7 Debugging Code

   

... is not easy in the MOO [more to come!!!]

The most important tricks are:

  • set verbs to +d ('@chmod #xxx:verb +d'). This will produce a trace when an errour is enountered.
  • Print out the arguments of verbs. Since lists or recursive lists are not printed you need a procedure for that: At TECFAMOO, use :report() with a single argument, .e.g. '@args'. Remember that errors (both syntax and logical) ocur because of non-anticipated or badly constructed data-structures !!
    You canOn every Lambda Core MOO you can also use this: <you>:tell($string_utils:from_value(args, 1, 10));
