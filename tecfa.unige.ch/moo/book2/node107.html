I want to program complex objects

 

No tutorials are avaible on this subject. You have to learn by looking at code.

One good idea for starters is to study the somewhat old-dated but still usefull Lambda CORE Programmer's Manual which explain how the objects in the original Lambda Core were programmed.

Next, you should browse a few interessting objects (e.g. $room or $thing) with an object Browser. At TECFAMOO, see section 16.3 on page for the using the E_WEB browser.
