1. Introduction

1-1 Clients MOO (Moo-Clients)

Pour travailler avec MOO, il est très fortement conseillé d'utiliser un client spécialisé. Il est difficile de travailler avec une connexion telnet.

1-1.1 Sous Emacs: le client "rmoo"

  • M-x rmoo

    • Vous propose le choix d'un Moo et vous interroge éventuellement sur votre nom + mot de passe. La liste des MOOs disponibles et stockée dans le fichier ~/.rmoo_worlds

  • c-c c-q Quitter une séance MOO

  • M-p Répéter (ou plutôt reafficher) la commande précédente

1-1.2 Muddweller sur MAC

  • Utilisez les menus pour vous connecter

  • Tapez vos commandes dans la toute petite fenêtre en bas du client.

1-1.3 (tf) Tinyfugue sous Unix

  • Tapez "tf <adresse> <no porte>"

  • la commande Unix "tecfamoo" lance tf pour le TecfaMOO.

  • /quit pour déconnecter

  • /visual on sépare la ligne input du reste

  • /help donne la liste des commandes
