6. La construction (Building)

6-5 Description plus avancée de "rooms" et sorties

En règle générale, on attache un certain nombre de messages aux entrées et sorties pour rendre le passage plus intéressant.

  • @messages imprime les messages attachables à un objet

    • Exemple:

    • (1) @messages me

    • Indique la liste des messages attachés à votre personnage

    • (2) @messages <exit>

    • Indique la liste des messages attachés à un "exit"

  • <@message-name> <object> is <"message .......">

    • Exemple:

    • (1) @leave n is "Vous vous promenez vers le nord ..... "

    • Ce message est vu par la personne qui emprunte une sortie. Utilisé à divers raisons, par exemple pour décrire une marche plus longue (par ex. "vous prenez le bus et .....") ou encore pour rendre la sortie intéressante (par ex "vous évitez de justesse de vous cogner en sortant...").

    • (2) @oleave n is "struggles towards the north"

    • Message destiné aux gens qui restent dans la même salle.

  1. Tous les noms des messages commençant par un "o" produisent un message qui met le nom de l'utilisateur au début du message

  2. Utilisation de variables: Dans un string vous mettre des variables qui seront substitués par des noms d'objets ou des promons (en Anglais). Faites un "help messages" pour des détails, ici juste le résumé en Anglais:


        %% => `%'  (just in case you actually want to talk about percentages).
    Names:
        %n => the player
        %t => this object (i.e., the object issuing the message,... usually)
        %d => the direct object from the command line
        %i => the indirect object from the command line
        %l => the location of the player
    Pronouns:
        %s => subject pronoun:          either `he',  `she', or `it'
        %o => object pronoun:           either `him', `her', or `it'
        %p => posessive pronoun (adj):  either `his', `her', or `its'  
        %q => posessive pronoun (noun): either `his', `hers', or `its'
        %r => reflexive pronoun:  either `himself', `herself', or `itself'
    General properties:
        %(foo) => player.foo 
        %[tfoo], %[dfoo], %[ifoo], %[lfoo]
               => this.foo, dobj.foo, iobj.foo, and player.location.foo
    Object numbers:
        %#  => player's object number
        %[#t], %[#d], %[#i], %[#l]
            => object numbers for this, direct obj, indirect obj, and location.

In addition there is a set of capitalized substitutions for use at the 
beginning of sentences.  These are, respectively, 

   %N, %T, %D, %I, %L for object names, 
   %S, %O, %P, %Q, %R for pronouns, and
   %(Foo), %[dFoo] (== %[Dfoo] == %[DFoo]),... for general properties
