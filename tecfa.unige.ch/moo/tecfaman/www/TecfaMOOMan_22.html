6-3 L'édition de texte

L'édition de textes associés aux objets est plus ou moins simple suivant le client MOO que vous utilisez. La commande `@notedit <quelque chose>' vous lance (par default) dans l'éditeur interne du MOO. Faites `look' une fois que vous êtes dedans. Il s'agit d'un simple éditeur qui vous permet d'éditer ligne par ligne.

Quelques commandes de base:

  • list Afficher les lignes d'un text

  • save Sauvegarder les modifications

  • quit Quitter l'éditeur

  • " Inséerer une ligne de texte à l'endroit où se trouve le pointeur

    • (1) "this is the first line

    • Insère une ligne "this is the first line" (à l'endroit où se trouve le pointeur)

  • ins Insérer une nouvelle ligne

    • (1) ins 3 "this text is nice

    • Insère une nouvelle ligne 3 avec le contenu "this text is nice".

  • del Détruire une ligne

    • (1) del 3

    • détruit la ligne No. 3

    • (2) del 4-10

    • détruit les lignes 4-10.

Pour les gens qui utilisent un client emacs, il est conseillé de taper `@edito +l' (une seule fois), dans la suite toute édition (mail, textes, etc.) vous ouvre un "buffer". Ne détruisez pas la première ligne, ni le "." à la fin et insérez votre texte entre ces lignes. Pour sauvegarder votre édition, tapez "Ctrl-C Ctrl-S").
