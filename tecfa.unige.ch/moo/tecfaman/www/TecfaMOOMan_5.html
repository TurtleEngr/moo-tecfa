1. Introduction

1-3 Aide (Help)

  • help Affiche une liste de "topics"

    • Arguments:

  • < topic> Affiche une introduction à un "topic" / liste de commandes disponibles

    • Exemple:

    • (1) help movement

    • Affiche une aide sur la navigation dans l'espace "MOO"

  • <commande> Affiche l'aide pour une commande

    • Exemple:

    • (1) help page

    • Affiche une aide pour la commande "page"

    • (2) help #<no objet>

    • Affiche l'aide pour un objet (si disponible)

    • (3)help #<no objet>:<verbe>

    • Affiche l'aide pour un verbe attaché à un objet
